## Hướng dẫn chụp ảnh màn hình

### Cách 1: Sử dụng Snipping Tool (Windows)
1. Mở Snipping Tool
2. Chọn "New" để tạo ảnh chụp mới
3. Chọn vùng màn hình muốn chụp
4. Lưu file với tên tương ứng vào thư mục `docs/chapter6/images/`

### Cách 2: Sử dụng PowerShell
```powershell
# Chụp toàn bộ màn hình
Add-Type -AssemblyName System.Windows.Forms
Add-Type -AssemblyName System.Drawing
$bounds = [System.Windows.Forms.Screen]::PrimaryScreen.Bounds
$bmp = New-Object System.Drawing.Bitmap $bounds.width, $bounds.height
$graphics = [System.Drawing.Graphics]::FromImage($bmp)
$graphics.CopyFromScreen($bounds.Location, [System.Drawing.Point]::Empty, $bounds.size)
$bmp.Save("C:\Users\Administrator\Desktop\SEAL-Hackathon-Management\docs\chapter6\images\filename.png")
```

### Cách 3: Sử dụng công cụ trực tuyến
- Sử dụng các công cụ như Lightshot, ShareX để chụp và lưu ảnh

## Lưu ý quan trọng
1. Tất cả ảnh phải được lưu vào thư mục: `docs/chapter6/images/`
2. Đặt tên file chính xác như danh sách trên
3. Ảnh nên rõ nét, hiển thị đầy đủ nội dung cần thiết
4. Định dạng ảnh: PNG (khuyến nghị) hoặc JPG
5. Kích thước ảnh: Width khoảng 1200-1400px để rõ nét khi in
